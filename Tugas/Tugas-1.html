Memahami Tipe Data#

Objek Data dan Tipe Atribut

Kumpulan data terdiri dari objek data. Objek data mewakili entitas—dalam penjualan database, objeknya mungkin pelanggan, item toko, dan penjualan; dalam database medis, the objek mungkin pasien; dalam database universitas, objeknya mungkin mahasiswa, profesor, dan kursus. Objek data biasanya dijelaskan oleh atribut. Objek data juga bisa disebut sebagai sampel, contoh, contoh, titik data, atau objek. Jika objek datanya adalah disimpan dalam database, mereka adalah tupel data. Artinya, baris database sesuai dengan objek data, dan kolom sesuai dengan atribut. Pada bagian ini, kami mendefinisikan atribut dan melihat berbagai jenis atribut.

PROBALITY_TABLE_VALUE = {
    "q" : (1,1),
    "r" : (1,0),
    "s" : (0,1),
    "t" : (0,0),
}

def get_series(data_sbmptn, idx, series):
    return data_sbmptn.loc[(idx), series]

def dissimilarity_dataset(data_sbmptn, series_index = [], series = []):
    first_series = get_series(data_sbmptn, series_index[0], series)
    second_series = get_series(data_sbmptn, series_index[1], series)
    dataset = pd.concat([first_series,second_series],axis=1)
    return dataset.T

Hitung hasil jarak jangkauan tipe data

def hasil_hitung_jarak(data_sbmptn, start_index = 0, last_index = 1):

    PROBALITY_VALUE = {"q" : 0, "r" : 0, "s" : 0, "t" : 0}

    column_range = data_sbmptn.shape[1]

    for column in range(column_range):
        for value in PROBALITY_TABLE_VALUE:
            item = list((tuple(data_sbmptn.loc[(start_index):(last_index), data_sbmptn.columns[column]]) == PROBALITY_TABLE_VALUE[value], value))
            if item[0] == True:
                if item[1] == "q":
                    PROBALITY_VALUE["q"] += 1
                if item[1] == "r":
                    PROBALITY_VALUE["r"] += 1
                if item[1] == "s":
                    PROBALITY_VALUE["s"] += 1
                if item[1] == "t":
                    PROBALITY_VALUE["t"] += 1

    return PROBALITY_VALUE
